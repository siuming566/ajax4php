ajax4php
========

A simple PHP AJAX Framework


Features
========

- MVC architecture
- AJAX support
- Webservice and WSDL generation
- Annotation
- RequestScope, ViewScope and SessionScope Model
- Implicit Navigation
- Layout Control

Requirements
============

- PHP 5.3+
- jQuery 1.7+
